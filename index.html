HTML Tags

HTML tags are the hidden keywords within a web page that define how your web browser must format and display the content.

Common HTML Tags

Tag Name Description
html Html document root tag The html root element will contain all html code
head Head Informtation about the webpage (metadata) like title, belongs in the head of the page
title Title A title can be added to the webpage in the head element using the title tag
body Body Any visible content should be placed within the opening and closing body tags
h1, h2, h3... Heading Headings and subheadings from h1-h6 are used to enlarge text
p Paragraph Specifies text
img Image Images can be added by linking to an existing source using the src attribute
a Anchor The anchor element is used to create hyperlinks. The links can direct towards other webpages, files on the same server, a location on the same page, or any other URL via the attribute href
div Division Short for 'division' or a container that divides the webpage into sections. Useful for grouping elements in your HTML document together

Semantic HTML Tags

Tag Name Description
header Header Defines the header for a document and provides basic structure, enables screen readers, and improves SEO
main Main Specifies the main content of a document, provides basic structure of the website
footer Footer Defines a footer for the document. Often contains links, contact info, etc.
section Section Defines elements in a document, such as chapters, headings, and any other areas of the doc with the same theme
article Article Holds content that makes sense on its own such as articles, blogs, comments, quotes, etc.
aside Aside Contains info that is related to the main content, but not required in order to understand the dominant information

HTML Attributes

Common HTML Attributes

Attribute Belongs To Description
class Global Attributes Specifies one of more classnames (refers to a class in a stylesheet)
id Global Attributes Can be called upon by CSS and Javascript to manipulate, format, and perform specific intructions on that element and that element only
style Global Attributes Specifies an inline CSS style for an element
href a, area, base, link The href determines the location the anchor element points to
src audio, embed, img, input, source, video, script, track Specifies the URL of a media file
type a, button, embed, input, link, menu, object, script, source, style Specifies the type of element, ie. when the input type = text, it specifies that text is to be inputted into a form
alt area, img, input Specifies an alternate text when the original element fails to display
target a, area, base, form Specifies the target for where to open the linked document or where to submit the form
